Problem 1

Before getting to the code, we need to derive some thoretical results.

We would like to choose $\bar{w}$ such that $y_t$ always has mean $1$. Do to that we need the first two moments of $w_t$.

\begin{align*} E[w_t] &= E \left[\bar{w} + \rho w_{t-1} + \epsilon_t \right] \\ &= \bar{w} + \rho E[w] \\ E[w] &= \frac{\bar{w}}{1 - \rho} \end{align*}

\begin{align*} var(w_t) &= var \left(\bar{w} + \rho w_{t-1} + \epsilon_t \right) \\ &= \rho^2 var(w_{t-1}) + var(\epsilon_t) \\ &= \rho^2 var(w_{t-1}) + \sigma_{\epsilon}^2 \\ var(w) &= \frac{\sigma_{\epsilon}^2}{1 - \rho^2} \end{align*}

This means that $w \sim N \left(\frac{\bar{w}}{1 - \rho}, \frac{\sigma_{\epsilon}^2}{1 - \rho^2} \right)$ and then $y \sim \log N \left(\frac{\bar{w}}{1 - \rho}, \frac{\sigma_{\epsilon}^2}{1 - \rho^2} \right)$.

$$ E_t[y_t] = \exp \left(\frac{\bar{w}}{1 - \rho} + \frac{\sigma_{\epsilon}^2}{2(1 - \rho^2)}\right) $$

We want $E[y] = 1$, which gives

\begin{align*} E_t[y_t] = 1 &= \exp \left(\frac{\bar{w}}{1 - \rho} + \frac{\sigma_{\epsilon}^2}{2(1 - \rho^2)}\right) \\ 0 &= \frac{\bar{w}}{1 - \rho} + \frac{\sigma_{\epsilon}^2}{2(1 - \rho^2)} \\ \bar{w} &= \frac{- \sigma_{\epsilon}^2 (1 - \rho)}{2 (1 - \rho^2)} \\ \bar{w} &= \frac{- \sigma_{\epsilon}^2 (1 - \rho)}{2 (1 - \rho)(1 + \rho)} \\ \bar{w} &= \frac{- \sigma_{\epsilon}^2}{2 (1 + \rho)} \end{align*}

function sim_moments(m::Markov; T=50000, burn=2000)
    """
    Given an object of type Markov, simulate a markov process, compute
    mean and variance of simulation, and compare to theoretical moments.
    """
    # Simulate and compute moments (throw out burn in period)
    
    sim = simulate(m, T)
    mu = mean(sim[burn:end])
    sig2 = var(sim[burn:end])

    # print nice string representation of moments
    E_str = "\t" * rpad(@sprintf("E[y]   = %.6f", mu), 17)
    E_str *= "\t" * @sprintf("Expected = %.6f", μ_y)
    E_str *= "\t" * @sprintf("Abs. Difference = %.6f", abs(mu - μ_y))
    V_str = "\n\t" * rpad(@sprintf("Var[y] = %.6f", sig2), 17)
    V_str *= "\t" * @sprintf("Expected = %.6f", σ2_y)
    V_str *= "\t" * @sprintf("Abs. Difference = %.6f", abs(sig2 - σ2_y))

    println(E_str * V_str)

    return mu, sig2, sim
end

Below I repeat part c, but with a value for the parameter $\rho$. This requires updating our value of $\sigma_{\epsilon}$ and, by extension, $\bar{w}$. We know that we would like the standard deviation to be the same as in part a, which was

$$\frac{\sigma_{\epsilon}}{\sqrt{1 - \rho^2}}.$$

The new standard deviation would be

$$\frac{\sigma_{\epsilon_d}}{\sqrt{1 - \rho_d^2}}.$$

So, the transformation we are looking for is

$$\sigma_{\epsilon_d} = \sigma_{\epsilon_d} \frac{\sqrt{1 - \rho_d^2}}{\sqrt{1 - \rho_a^2}}$$

It may seem counter-intuitive that when the borrowing constraint is looser, the average lifetime consumption of the agents is lower.

Let's do some more simulations to experiment with this result. I claim that one possible explaination is that agents are trying to maximize expected present discounted value of utility from consumption, not the expected value of average lifetime utilty from consumption. The distinction is subtle, but important. The less restrictive borrowing constraint allows agents to go on a consumption binge in the early periods of his life, when the discounted value of consumption is relatively higher than it is later in the agent's life. However, budget constraints still need to be satisfied, so in order to finance high early consumption, agents are willing to maintain a relatively lower consumption later in life, leading to lower average consumption over a long horizon.

@everywhere begin
function compute_psi(m::IFP, sim::Sim)
    # Use the values of y to get the values for w
    w = log(sim.y_it)
    c = sim.c_it
    eps_t = w[2:end, :] - 0.9 * w[1:end-1, :]
    ln_c = log(c)
    ln_c_diff = ln_c[2:end, :] - ln_c[1:end - 1, :]

    var_eps = mean(var(eps_t, 1))
    cov_c_eps = 0.0
    for i=1:size(eps_t, 2)
        cov_c_eps += cov(ln_c_diff[:, i], eps_t[:, i])
    end

    cov_c_eps /= size(eps_t, 2)

    return 1.0 - cov_c_eps / var_eps
end

end  # everywhere

The problem states the following:

Argue that, based on the value of $\psi$ the model where the borrowing limit is looser displays more consumption insurance.

As can be seen above, the results from my simluation are that the model with the no borrowing constraint displays more consumption insurance. Although it is not what Gianluca was expecting, I feel that this result is intuitive for the following. In the model with no borrowing constraint agents will be willing to purchase more insurance in order to save against a possible future where a bad realization of shocks hits (meaning they have low income) an they are unable to borrow to finance consumption (because of the no borrowing constraint). Agents that have a looser borrowing constraint do not have quite as strong as a motive for precautionary savings, because if they find themselves in the position we just described, they will be able to borrow to fund consumption.
